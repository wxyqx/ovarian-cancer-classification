## 一、产品概述

### 1.1 产品定位

一款基于AI的个性化学习管理系统，通过智能分析、数据可视化和游戏化设计，帮助用户高效管理学习进度、提升学习动力。

### 1.2 核心价值

- **个性化学习** - AI驱动的智能任务分配
- **可视化进度** - 直观的进度追踪系统
- **游戏化体验** - 成就系统与打卡激励
- **AI智能分析** - 深度学习画像与建议

### 1.3 目标用户

- 学生群体（K12、大学生）
- 职场学习者
- 语言学习爱好者

---

## 二、设计原则

1. **清晰直观** - 信息层级分明，一目了然
2. **游戏化激励** - 多邻国风格的进度系统和成就体系
3. **AI驱动** - 智能任务分配和个性化建议
4. **渐进解锁** - 线性学习路径，确保知识连贯性
5. **数据驱动** - 基于学习数据的智能分析

#### 5.1.3 关卡按钮设计

**状态定义**:

| 状态 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 未解锁 | 🔒 | 灰色 | 需要完成前置关卡 |
| 已解锁 | ○ | 蓝色边框 | 可点击进入 |
| 进行中 | ▶ | 渐变填充 | 显示进度条 |
| 已完成 | ✓ | 绿色 | 已完成该关卡 |

**交互规则**:

```
线性解锁机制：
第1关(完成) → 第2关(解锁) → 第3关(解锁) → 第4关(解锁) → ...

规则：
1. 必须按顺序完成关卡
2. 完成当前关卡后，下一关自动解锁
3. 点击已解锁关卡进入学习
4. 未解锁关卡点击无反应
```

**按钮样式详解**:

**未解锁状态**:

```
┌──────────────────────┐
│  🔒  第5关 日常对话   │
│      [未解锁]        │
└──────────────────────┘
- 背景: #E0E0E0 (灰色)
- 文字: #9E9E9E (浅灰)
- 禁用点击
```

**已解锁状态**:

```
┌──────────────────────┐
│  ○  第5关 日常对话   │

└──────────────────────┘

- 光标: pointer
```

#### 5.2.2 AI任务生成逻辑

**任务类型分类**:

1. **每日任务 (Daily)**
   - 单词背诵
   - 听力练习
   - 口语对话
   - 阅读理解
   - 写作练习

2. **每周任务 (Weekly)**
   - 专题学习
   - 复习巩固
   - 能力测试
   - 小组活动

3. **每月任务 (Monthly)**
   - 阶段考核
   - 学习报告
   - 目标复盘
   - 技能认证

**AI生成规则**:

```
- 根据用户历史学习数据
- 分析薄弱环节
- 结合学习目标
- 生成个性化任务
```

1. 点击切换内容区域
2. 当前选中项高亮 (主色填充)
3. 未选中项灰色
4. 切换时内容区域平滑过渡 (300ms)
5. 底部导航固定在屏幕底部

### 6.2 关卡按钮交互

**线性解锁交互流程**:

**用户点击第5关(未解锁)**:

1. 按钮抖动动画
2. 提示"请先完成第4关"
3. 自动滚动到第4关位置
4. 高亮第4关

**用户完成第4关**:

1. 第4关变为"已完成"状态
2. 第5关解锁动画 (锁打开)
3. 第5关变为"已解锁"状态
4. 触发成就检查

**用户点击第5关(已解锁)**:

1. 按钮缩放动画 (点击效果)
2. 跳转至第5关学习页面

### 6.3 任务卡片交互

**任务卡片交互流程**:

```
未开始 → 进行中 → 已完成
```

**状态转换动画**:

1. 勾选完成：打钩动画 + 绿色渐变
2. 进度更新：进度条平滑动画
3. 新任务添加：从右侧滑入
4. 任务过期：渐隐 + 红色提示
