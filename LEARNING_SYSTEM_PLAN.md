# 学习反馈系统 — 规划设计文档

## 1. 系统概述

一个多学科学习反馈系统，帮助学习者记录、评估和优化自己的学习过程。覆盖学科：

| 学科 | 聚焦方向 |
|------|----------|
| 英语 | 词汇、语法、听说读写、考试准备 |
| 心理学 | 理论记忆、案例分析、实验理解 |
| 认知学 | 认知模型、学习方法论、元认知训练 |
| 商业 | 商业模式、案例分析、战略思维 |

## 2. 核心功能规划

### 第一阶段：基础反馈（当前原型）

```
┌─────────────────────────────────────────────┐
│              📚 学习反馈系统                  │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│  │ 英语     │ │ 心理学   │ │ 认知学   │ ... │
│  ├──────────┤ ├──────────┤ ├──────────┤     │
│  │ 今日学习 │ │ 今日学习 │ │ 今日学习 │     │
│  │ 反馈打分 │ │ 反馈打分 │ │ 反馈打分 │     │
│  │ 查看记录 │ │ 查看记录 │ │ 查看记录 │     │
│  └──────────┘ └──────────┘ └──────────┘     │
├─────────────────────────────────────────────┤
│  📊 总体进度  │  🔔 提醒  │  ⚙️ 设置        │
└─────────────────────────────────────────────┘
```

- **今日学习打卡** — 选学科 → 写学了什么 → 自评掌握度（1-5）
- **反馈记录** — 按学科查看历史记录
- **总览面板** — 看到所有学科的近期状态

### 第二阶段：智能反馈

- AI 分析学习记录，给出改进建议
- 自动识别薄弱环节
- 个性化学习路径推荐

### 第三阶段：深度整合

- 间隔重复提醒（艾宾浩斯曲线）
- 学习统计图表
- 多设备同步（数据库）

## 4. 数据模型

```
Subject（学科）
  ├── id, name, color
  └── Entry（学习记录）
        ├── id, date, content, rating(1-5), notes
        └── Feedback（AI 反馈，二期）
```

## 5. 当前原型范围
